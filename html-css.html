Web Development Unit

What is Web Development

Web development is the work involved in developing the front end and back end of a web site. This is usually done for a client who hired a web developer to make a website, social media service, corporate data base or application for them.

Web development includes both front-end programming and back-end programming. Front-end programming, also known as the client side, is what a person who visits the cite sees. This includes using the languages HTML (markup language), CSS (style sheet language) and JavaScript (scripting language). Although a site can be created with only front-end programming, if one wants to collect data such as payment methods, survey results, and any other types of user input, back-end programming is needed. Back-end programming is the side of the website that works with servers, and stores input from users who access the website. Back-end work requires programming languages such as Java, C++, Python, etc.
